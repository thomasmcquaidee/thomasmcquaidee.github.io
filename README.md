# [tmcq.io](https://tmcq.io)

My personal website, built with Jekyll

## Requirements

- [Bundler](https://bundler.io)
- [Jekyll](https://jekyllrb.com)

## Installation

```bash
bundle install
```

## Build

```bash
bundle exec jekyll build
```

## Preview

```bash
bundle exec jekyll serve
```
